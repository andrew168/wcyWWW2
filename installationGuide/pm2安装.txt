设置PM2，
* npm install pm2 -g
(对于win 10， 需要-f, -f避免安装mac专用的fsevents模块，可否替换为 --no-optional？？)

* 设置PM2_HOME环境变量为自定义的目录：例如：
PM2_HOME=D:/etc/pm2
供pm2 save保存数据
* pm2 init //(先初始化，生成ecosystem.config.js， 如果没有的话）
* pm2 start ecosystem.config.js --env production
* pm2 reload app //热重新加载（reload至少保证1个实例运行，从而确保有服务）
* pm2 kill
* pm2 ls

// 更多操作
* pm2 start your-app.js --name 新名字
* pm2 start your-app.js --name 新名字 -i max //并行(最多达到CPU数目）
* pm2 start your-app.js --name 新名字 -i 20 //并行(20个）
* pm2 stop app (只kill process, 但保留在process list中)
* pm2 start app
* pm2 restart app //restart：关闭并且马上重启，但是会短暂无服务。

* pm2 resurrect
* pm2 delete app
* pm2 save
* pm2 monit
